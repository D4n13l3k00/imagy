# imagy

## imagy - in-terminal image viewer written in Rust

![im](https://i.imgur.com/WmUADq1.png)

### Installation

```bash
cargo install --git https://github.com/D4n13l3k00/imagy
```
### Shortcuts

```text
Q | ESC -> Exit
      I -> Minimal image info
      R -> Rerender image
```
